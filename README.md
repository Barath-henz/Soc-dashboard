# 🔐 SOC Monitoring Dashboard

A full-stack Security Operations Center (SOC) dashboard for real-time threat detection, log analysis, and security monitoring.

---

## 🚀 Features

* 🔐 User Authentication (Admin Panel)
* 📡 Real-time alerts using WebSockets
* 🌍 IP Geolocation tracking
* 🤖 AI-based anomaly detection
* 📧 Email alert system
* 📊 Interactive dashboard (Chart.js)
* 🐳 Dockerized deployment

---

## 🏗️ Tech Stack

* Backend: Python (Flask, SocketIO)
* Frontend: HTML, CSS, JavaScript
* Database: SQLite
* DevOps: Docker, Docker Compose
* Visualization: Chart.js

---

## ⚙️ Setup (Local)

```bash
git clone https://github.com/Barath-henz/Soc-dashboard.git
cd Soc-dashboard
docker compose up --build
```

Open:

* Frontend → http://localhost:80
* Backend → http://localhost:5000
